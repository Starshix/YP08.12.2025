Проект GearLock реализует все требования по трём частям. Бэкенд: настроен Django фреймворк с подключением к SQLite базе данных по спроектированной схеме. Реализовано API через Django Views для всех ключевых операций: аутентификация и регистрация пользователей, CRUD для товаров и заказов, эндпоинты для административных функций. Система ролей включает четыре уровня: покупатель, менеджер, контент-менеджер и администратор с соответствующими правами доступа.

Фронтенд: настроен на Django Templates с Bootstrap 5. Созданы компоненты для всех страниц: главная, каталог товаров, карточка товара, корзина, оформление заказа, личный кабинет, панели управления.  Организована навигация через меню и хлебные крошки. Созданы валидируемые формы: регистрация, вход, оформление заказа, управление товарами.

Административная панель: выделены отдельные разделы для каждой роли. Реализовано управление пользователями с фильтрацией по ролям. Модерация контента включает управление товарами, категориями, заказами с изменением статусов. Инструменты аналитики: дашборды с количеством заказов, выручкой, клиентами, статистикой товаров. Доступ к разделам админки строго контролируется через систему ролей и декораторы доступа.

Чтобы запустить проект, нужно:

1. Скачать проект из этого репозиторий.
2. Открыть папку YP08.12.2025-main в VS Code
3. Прописать в терминале pip install django
4. Запустить проект python manage.py runserver

Если будет выдавать ошибки python -m venv venv
